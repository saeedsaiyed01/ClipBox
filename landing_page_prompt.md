# ClipBox Landing Page Prompt

## Product Overview
Create a modern, professional landing page for **ClipBox** - a web-based video processing application that transforms videos with custom backgrounds, aspect ratios, and styling effects.

## Product Description
ClipBox is a full-stack video processing application that allows users to upload videos and transform them with professional-grade effects through an intuitive web interface. Users can customize backgrounds, adjust aspect ratios, control zoom levels, and add border radius effects with real-time preview and one-click export.

## Key Features to Highlight

### Core Functionality
- **Video Upload & Processing**: Drag & drop video upload with instant preview
- **Custom Backgrounds**: Solid colors, gradients, and image backgrounds
- **Aspect Ratio Control**: Support for 1:1, 4:5, 9:16, and 16:9 ratios
- **Zoom Control**: Scale videos from 50% to 200%
- **Border Radius**: Add rounded corners with customizable radius
- **Real-time Preview**: See changes instantly as you make adjustments
- **One-Click Export**: Download processed videos with professional quality

### Technical Excellence
- **Modern Tech Stack**: Built with Next.js, React, TypeScript
- **Asynchronous Processing**: Powered by Redis and BullMQ job queues
- **FFmpeg Integration**: Professional video processing capabilities
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile
- **Progress Tracking**: Real-time processing status with visual progress bars
- **Production Ready**: Environment configuration, logging, and error handling

## Design Guidelines

### Color Scheme
- **Primary**: Dark theme with deep blacks (#09090b)
- **Secondary**: Amber/Yellow accents (#F59E0B, #FBBF24)
- **Text**: White (#FFFFFF) and gray variants (#A1A1AA, #71717A)
- **Background**: Dark gradients and glass morphism effects

### Typography
- **Headings**: Modern, bold sans-serif (Inter or similar)
- **Body**: Clean, readable sans-serif
- **Accent**: For CTAs and highlights

### Visual Style
- **Modern glassmorphism** with backdrop blur effects
- **Subtle gradients** and smooth animations
- **Clean icons** with consistent style
- **Professional photography** and mockups
- **Video previews** and interactive elements

### Layout
- **Full-width sections** with proper spacing
- **Mobile-first responsive** design
- **Grid-based layouts** for features
- **Sticky navigation** with smooth scrolling

### Interactive Elements
- **Hover effects** on buttons and cards
- **Smooth transitions** and micro-animations
- **Loading states** and progress indicators
- **Video previews** with play/pause functionality
